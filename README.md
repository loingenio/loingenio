# Lo Ingenio — sitio web

Landing page estática lista para GitHub Pages.

## Publicar

1. Crea un repositorio en GitHub y sube todos los archivos del paquete a la raíz, incluidos `favicon.png` y los dos archivos de logo.
2. En **Settings → Pages**, selecciona **Deploy from a branch**, la rama `main` y la carpeta `/ (root)`.
3. En el proveedor de tu dominio crea un registro **CNAME** para `www` apuntando a `TU-USUARIO.github.io`.
4. En GitHub Pages confirma `www.loingenio.org` como dominio personalizado y activa **Enforce HTTPS** cuando quede disponible.

## Antes de publicar

- Reemplaza los testimonios marcados como placeholders.
- Actualiza enlaces de redes sociales (`href="#"`).
- Confirma el correo `hola@loingenio.org` y las fechas de cursos/eventos.
- Conecta los botones de recursos a sus archivos o formularios reales.

El sitio usa Google Fonts. Si prefieres evitar dependencias externas, descarga DM Sans y Manrope y enlázalas localmente.
